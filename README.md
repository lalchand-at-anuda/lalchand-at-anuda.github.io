## Local Development

```bash
# Clone the repository
git clone https://github.com/lalchand-at-anuda/lalchand-at-anuda.github.io.git

# Navigate to directory
cd lalchand-at-anuda.github.io

# Start a local server (any of these)
python3 -m http.server 8000
# or
npx http-server
# or
php -S localhost:8000

# Open http://localhost:8000 in your browser
```

## Customization

To customize this portfolio for your own use:

1. Update personal information in `index.html`
2. Modify color scheme in `styles.css` (CSS custom properties at the top)
3. Add your own projects and skills
4. Update contact information
5. Replace typing animation phrases in `script.js`
