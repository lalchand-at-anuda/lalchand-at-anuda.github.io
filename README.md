## ğŸ’» Local Development

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional but recommended)

### Quick Start

1. **Clone the repository**
```bash
git clone https://github.com/lalchand-at-anuda/lalchand-at-anuda.github.io.git
```

2. **Navigate to directory**
```bash
cd lalchand-at-anuda.github.io
```

3. **Start a local server** (choose one)
```bash
# Python 3
python3 -m http.server 8000

# Node.js
npx http-server

# PHP
php -S localhost:8000
```

4. **Open in browser**
```
http://localhost:8000
```

## ğŸ¨ Customization

To customize this portfolio for your own use:

1. **Personal Information** - Update `index.html`
   - Name, title, and description
   - Contact details (phone, email, LinkedIn)
   - Profile photo path

2. **Color Scheme** - Modify `styles.css`
   - CSS custom properties at the top of the file
   - Primary and secondary colors
   - Gradient values

3. **Projects** - Add your projects in `index.html`
   - Project title and description
   - Technologies used
   - Project tags

4. **Skills** - Update skill badges in `index.html`
   - Add/remove technologies
   - Categorize by Frontend/Backend/DevOps

5. **Typing Animation** - Edit `script.js`
   - Update the phrases array with your roles
   - Adjust typing speed if needed

6. **Resume** - Replace the PDF file
   - Update the filename in the navigation link
